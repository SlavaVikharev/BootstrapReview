"А откудова эти классы брать?"

У Bootstrap есть сайт (о_О)

https://getbootstrap.com/

На этом сайте есть разделы "CSS", "Components"

На них расположено всё, что можно использовать

Bootstrap нужно подключить

Так как Bootstrap - это просто файл .css, то добавляем ссылку на него и радуемся

<link rel="stylesheet" href="css/bootstrap.min.css">
